In cookie the data can be set to expire at a certain time. and Cookies are a good choice 
for storing data that should not be persisted for a long time, such as session IDs.
but in Local storage the stored data is not deleted when the browser is closed. and 
Local storage is useful for storing data that the user will need to access later,
such as offline data. while In Session storage The data is stored only for the session 
and will be deleted when the browser is closed. and Session storage is a great way to improve
the performance of your web applications.
